# FlixList (How to Run Client and API)

## Front End

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.0.

### Client server

Go to client directory and run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

---------------------------------------------------------

## Back End

This project was built with NodeJS, Express, MongoDB, Mongoose, and several other npm packages.

### Api server

Go to api directory and run `nodemon` or `npm run start`. The server will run on `http://localhost:2020/`.
